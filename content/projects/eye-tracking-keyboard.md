## Eye Tracking Keyboard

An innovative eye tracking keyboard that works on any device with a webcam! This project enables hands-free typing through gaze detection technology.

### Features

- Works with any standard webcam
- Real-time eye tracking and gaze detection
- Virtual keyboard interface
- Hands-free typing capability

### Applications

- Accessibility tool for users with mobility impairments
- Hands-free computing in sterile environments
- Alternative input method for various use cases
